📝 Linux Crontab 定时任务笔记总结

🔍 核心概念

crontab 是 Linux/Unix 系统中用于设置周期性执行任务的工具，通过守护进程 cron 在后台自动运行指定命令，常用于：

  • ✅ 定时备份数据库
  • ✅ 日志轮转清理
  • ✅ 系统监控告警
  • ✅ 自动化脚本调度

⚙️ 常用命令速查

命令 功能说明 使用示例
crontab -e ✏️ 编辑当前用户的定时任务 crontab -e
crontab -l 📋 列出当前用户的定时任务 crontab -l
crontab -r 🗑️ 删除当前用户的所有定时任务 crontab -r

💡 小贴士：-e 选项默认使用系统配置的编辑器（如 vim/nano），可通过 export EDITOR=vim 临时修改。
